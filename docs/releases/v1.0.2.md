# v1.0.2

This release aligns the Laravel package and Stagehand runtime release surfaces after the navigation-policy flags added post-v1.0.1.

## Fixed

- Prevents the Laravel installer from accepting a stale Stagehand binary that does not expose the serve flags required by the current package.
- Updates the default Stagehand runtime release to v1.0.2 so generated release assets match the package command surface.
- Namespaces Stagehand Chromium profiles per runtime process so stale browser children from a crashed or force-stopped runtime cannot lock the next process out of local PDF rendering.
