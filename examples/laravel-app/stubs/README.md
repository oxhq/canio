# Canio Example App

This application is generated by `examples/laravel-app/create-project.sh` and exists to exercise the local `oxhq/canio` package end to end.

It is wired as a path repository to the package source in this repo and ships with:

- invoice preview and dispatch routes
- a JavaScript probe document for runtime capability checks
- optional Canio Cloud smoke routes
- example ops panel configuration for local validation

Primary routes:

- `/invoices/preview`
- `/invoices/dispatch`
- `/probes/javascript`
- `/probes/javascript/preview`
- `/canio/cloud/sync/preview`
- `/canio/cloud/managed/preview`
- `/canio/ops`

For bootstrap and workflow details, see `../README.md`.
For development commands, tests, and benchmarks, see `docs/development.md` in the Canio repository root.
